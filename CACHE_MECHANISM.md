# 一般使用者書單雲端快取機制

## 概述
本系統實現了一個智能的書單快取機制，避免一般使用者每次打開網頁、切換列表、搜尋、切換圖示顯示時都重新從雲端下載書單。

## 快取邏輯

### 對一般使用者的影響
1. **首次進入網站（今天）**
   - 系統會檢查今天是否已下載過書單
   - 如果還沒下載，會從雲端下載最新書單
   - 將書單保存到 localStorage 快取中
   - 記錄下載日期和版本信息

2. **之後的訪問（同一天內）**
   - 系統檢查是否已有今天的快取
   - 檢查雲端版本是否有更新
   - 如果沒有更新，直接使用快取的書單
   - 如果有更新，重新下載最新書單

3. **日常操作（切換列表、搜尋、借閱等）**
   - 這些操作都不會觸發重新下載
   - 系統會直接使用已加載的書單
   - 提升系統響應速度和用戶體驗

4. **使用者手動刷新**
   - 點擊「重新整理書單」按鈕
   - 系統清除快取，強制從雲端重新下載

### 對管理員的影響
1. **新增/編輯/刪除書籍**
   - 操作後自動更新版本時間戳
   - 將更新同步到 Google Sheets

2. **版本管理**
   - 每次操作後都會更新版本信息
   - 一般使用者下次訪問時會偵測到版本變更
   - 自動重新下載最新書單

## localStorage 鍵值

系統使用以下 localStorage 鍵值來管理快取：

| 鍵值 | 說明 | 例值 |
|------|------|------|
| `lib_books_v1` | 書籍列表（原有） | JSON 數組 |
| `lib_borrowed_v1` | 借閱記錄（原有） | JSON 數組 |
| `lib_cache_version` | 書單版本時間戳 | `2026-06-04T10:30:00.000Z` |
| `lib_cache_download_date` | 上次下載日期 | `2026-06-04` |

## 新增的功能

### 1. 快取系統函數
- `initCacheSystem()` - 初始化快取系統
- `checkShouldUpdateBookList()` - 檢查是否需要更新
- `saveBookListCache()` - 保存快取
- `loadBookListCache()` - 加載快取
- `updateBookListVersion()` - 更新版本
- `clearBookListCache()` - 清除快取
- `getCacheStatus()` - 獲取快取狀態

### 2. 新增使用者按鈕
- 「重新整理書單」按鈕
- 位置：主控制面板
- 權限：任何使用者都可使用（不需要管理員權限）
- 功能：清除快取，強制重新下載書單

## 流程圖

### 網站首次載入
```
頁面載入
  ↓
檢查是否是今天第一次進入
  ├─ 是 → 從雲端下載書單 → 保存快取 → 顯示書單
  └─ 否 → 檢查版本是否更新
          ├─ 有更新 → 從雲端下載 → 更新快取 → 顯示書單
          └─ 無更新 → 從快取加載 → 顯示書單
```

### 管理員新增書籍
```
點擊「新增館藏」
  ↓
填寫書籍信息
  ↓
提交表單
  ↓
更新書單版本
  ↓
同步到 Google Sheets
  ↓
保存成功消息
```

### 使用者手動刷新
```
點擊「重新整理書單」
  ↓
清除本地快取
  ↓
從雲端重新下載書單
  ↓
更新 UI
  ↓
顯示成功消息
```

## 效能改進

### 改進前
- 每次進入網站都要從雲端下載完整書單
- 切換列表、搜尋、借閱都可能觸發重新下載
- 網路較慢時體驗不佳

### 改進後
- 同一天內只需下載一次（或版本更新時）
- 切換列表、搜尋、借閱等操作瞬間完成
- 大幅減少雲端 API 請求
- 提升使用者體驗

## 測試建議

1. **首次訪問測試**
   - 開啟瀏覽器開發者工具 → Application → Local Storage
   - 查看 `lib_cache_download_date` 是否設置為今天

2. **同日再次訪問測試**
   - 刷新頁面
   - 觀察控制台，應該看到 `[快取策略] 成功加載快取書籍...`

3. **版本更新測試**
   - 管理員新增書籍
   - 觀察 `lib_cache_version` 是否已更新

4. **手動刷新測試**
   - 點擊「重新整理書單」
   - 觀察 localStorage 中的快取是否清除

## 故障排查

### 快取不工作？
1. 檢查 localStorage 是否被瀏覽器禁用
2. 檢查隱私模式是否開啟
3. 檢查 localStorage 容量是否已滿

### 版本不更新？
1. 檢查管理員是否正確新增/編輯/刪除書籍
2. 檢查 Google Sheets 同步是否成功
3. 檢查 `lib_cache_version` 是否有時間戳

### 快取過期？
- 系統設計為按天快取，不同天期會自動更新
- 需要立即更新可點擊「重新整理書單」按鈕
